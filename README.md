# Project Portal

## Overview

Project Portal is a solo-developed 3D first-person puzzle game inspired by physics-based puzzle mechanics. Players use a portal gun to navigate two puzzle rooms by manipulating portals, interacting with weighted cubes, avoiding automated turrets, and solving environmental challenges to progress.

The project was developed in Unreal Engine 5 as a rapid prototype to demonstrate gameplay programming, puzzle design, and player movement systems.

## Technical Challenges

One of the biggest technical challenges was implementing reliable portal mechanics while preventing visual and gameplay issues caused by thin level geometry. Careful level design and wall thickness were required to ensure portals behaved consistently without allowing unintended interactions.

Another challenge was balancing development speed with maintainable code. Since the project was developed as a rapid prototype in approximately one week, I adopted a Blueprint-focused workflow to quickly implement gameplay systems while maintaining a clean and expandable project structure.
